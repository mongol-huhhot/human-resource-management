## 扶養控除について

<扶養控除額合計> ::= SUM(<扶養控除額> for each <親族>)

<扶養控除額> ::=
      0
    | 380000
    | 480000
    | 580000
    | 630000

<扶養親族> ::=
    <判定日時点で判定>
    AND <配偶者以外の親族等>
    AND <生計を一にする>
    AND <扶養所得要件を満たす>
    AND <事業専従者でない>


<判定日時点で判定> ::=
    原則 <tax_year>-12-31 時点
    | 納税者が年途中で死亡した場合は死亡時点
    | 納税者が年途中で出国した場合は出国時点

<配偶者以外の親族等> ::=
      <6親等内の血族>
    | <3親等内の姻族>
    | <里子>
    | <市町村長から養護を委託された老人>

<扶養所得要件を満たす> ::=
    <親族.合計所得金額> <= <扶養所得上限額>

<扶養所得上限額> ::=
      580000    ; tax_year >= 2025, 令和7年分以後
    | 480000    ; 2020 <= tax_year <= 2024, 令和2年分〜令和6年分
    | 380000    ; tax_year <= 2019, 令和元年分以前

<事業専従者でない> ::=
    NOT <青色申告者の事業専従者として年間を通じ給与支払あり>
    AND NOT <白色申告者の事業専従者>


<控除対象扶養親族> ::=
    <扶養親族>
    AND <年齢16歳以上>
    AND <居住者または非居住者控除対象>

<年齢16歳以上> ::=
    <年末年齢> >= 16

 <居住者または非居住者控除対象> ::=
      <居住者>
    | <非居住者控除対象>   

<非居住者控除対象> ::=
      <年末年齢> >= 16 AND <年末年齢> < 30
    | <年末年齢> >= 70
    | <年末年齢> >= 30 AND <年末年齢> < 70 AND <非居住者30歳以上70歳未満要件>

<非居住者30歳以上70歳未満要件> ::=
      <留学により非居住者>
    | <障害者>
    | <生活費または教育費として年間380000円以上の支払あり>

<非居住者書類要件OK> ::=
      <年齢16以上30未満または70以上>
        AND <親族関係書類あり>
        AND <送金関係書類あり>
    | <年齢30以上70未満かつ留学>
        AND <親族関係書類あり>
        AND <送金関係書類あり>
        AND <留学ビザ等書類あり>
    | <年齢30以上70未満かつ障害者>
        AND <親族関係書類あり>
        AND <送金関係書類あり>
    | <年齢30以上70未満かつ38万円以上送金>
        AND <親族関係書類あり>
        AND <38万円送金書類あり>

<扶養控除額> ::=
      0
        IF NOT <扶養親族>

    | 0
        IF NOT <控除対象扶養親族>

    | 580000
        IF <同居老親等>

    | 480000
        IF <老人扶養親族>

    | 630000
        IF <特定扶養親族>

    | 380000
        IF <一般の控除対象扶養親族>

<特定扶養親族> ::=
    <控除対象扶養親族>
    AND <年末年齢> >= 19
    AND <年末年齢> < 23

<老人扶養親族> ::=
    <控除対象扶養親族>
    AND <年末年齢> >= 70

<同居老親等> ::=
    <老人扶養親族>
    AND <本人または配偶者の直系尊属>
    AND <本人または配偶者と同居を常とする>
    AND NOT <老人ホーム入所>    

<一般の控除対象扶養親族> ::=
    <控除対象扶養親族>
    AND NOT <特定扶養親族>
    AND NOT <老人扶養親族>


<年末調整.扶養控除額合計> ::=
    SUM(
        CASE
            WHEN NOT <扶養親族> THEN 0
            WHEN NOT <控除対象扶養親族> THEN 0
            WHEN <非居住者> AND NOT <非居住者書類要件OK> THEN 0
            WHEN <同居老親等> THEN 580000
            WHEN <老人扶養親族> THEN 480000
            WHEN <特定扶養親族> THEN 630000
            ELSE 380000
        END
    )

<扶養控除対象外だが特定親族特別控除候補> ::=
    <配偶者以外の親族等>
    AND <生計を一にする>
    AND <年末年齢> >= 19
    AND <年末年齢> < 23
    AND <親族.合計所得金額> > 580000
    AND <親族.合計所得金額> <= 1230000
    AND <事業専従者でない>


<判定日> ::=
      <tax_year>-12-31
    | <本人.年途中死亡日>
    | <本人.年途中出国日>

<年末年齢> ::=
    age_at(<対象者.生年月日>, <判定日>)

<扶養所得上限額> ::=
      580000  IF <tax_year> >= 2025
    | 480000  IF 2020 <= <tax_year> <= 2024
    | 380000  IF <tax_year> <= 2019

<給与収入のみ扶養上限額> ::=
      1230000 IF <tax_year> >= 2025
    | 1030000 IF <tax_year> <= 2024

<配偶者以外の親族等> ::=
      <6親等内の血族>
    | <3親等内の姻族>
    | <里子>
    | <市町村長等から養護委託された老人>

<生計同一> ::=
    <本人と生計を一にする>

<事業専従者でない> ::=
    NOT <青色事業専従者として年間を通じ給与支払あり>
    AND NOT <白色事業専従者>

<扶養親族共通条件> ::=
    <配偶者以外の親族等>
    AND <生計同一>
    AND <対象者.合計所得金額> <= <扶養所得上限額>
    AND <事業専従者でない>


MODULE <扶養控除>

INPUT ::=
    <tax_year>
    <本人>
    <親族リスト>

OUTPUT ::=
    <扶養控除額合計>
    <扶養控除明細リスト>


<扶養控除対象者> ::=
    <扶養親族共通条件>
    AND <年末年齢> >= 16
    AND <居住者または非居住者控除対象>

<居住者または非居住者控除対象> ::=
      <対象者.居住区分> = "resident"
    | <非居住者扶養控除対象>

<非居住者扶養控除対象> ::=
      16 <= <年末年齢> AND <年末年齢> < 30
    | 70 <= <年末年齢>
    | 30 <= <年末年齢> AND <年末年齢> < 70 AND <非居住者30歳以上70歳未満要件>

<非居住者30歳以上70歳未満要件> ::=
      <留学により非居住者>
    | <障害者>
    | <生活費または教育費として年間380000円以上支払あり>


<特定扶養親族> ::=
    <扶養控除対象者>
    AND 19 <= <年末年齢>
    AND <年末年齢> < 23

<老人扶養親族> ::=
    <扶養控除対象者>
    AND 70 <= <年末年齢>

<同居老親等> ::=
    <老人扶養親族>
    AND <本人または配偶者の直系尊属>
    AND <本人または配偶者と同居を常とする>
    AND NOT <老人ホーム等入所>

<一般の控除対象扶養親族> ::=
    <扶養控除対象者>
    AND NOT <特定扶養親族>
    AND NOT <老人扶養親族>


<扶養控除額> ::=
      0      IF NOT <扶養控除対象者>
    | 580000 IF <同居老親等>
    | 480000 IF <老人扶養親族>
    | 630000 IF <特定扶養親族>
    | 380000 IF <一般の控除対象扶養親族>

<扶養控除額合計> ::=
    SUM(<扶養控除額> FOR EACH <親族> IN <親族リスト>)
